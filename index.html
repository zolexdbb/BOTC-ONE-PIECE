🗺️ La map
  • Les joueurs évoluent sur une map Minecraft dédiée. Chacun possède sa propre chambre où il se retire la nuit.
  • Une salle du conseil centrale sert de point de rassemblement pour les annonces et les votes.
  • Durant la phase de jour, les joueurs sont libres de se déplacer sur la map pour discuter en privé ou en groupe.
🌙 Phase de nuit
  • Quand la nuit tombe, tous les joueurs regagnent leur chambre et ne doivent pas en sortir.
  • Le MJ se déplace de chambre en chambre pour visiter les joueurs un par un et leur transmettre les informations liées à leur rôle.
  • Les actions nocturnes (meurtre de Sengoku, poison d'Aramaki, protection de Kuma…) sont résolues en coulisse par le MJ durant ces visites.
  • Un joueur empoisonné par Aramaki peut recevoir de fausses infos ou voir son pouvoir annulé — sans le savoir.
  • Ordre indicatif des visites — première nuit : Marines (révélation mutuelle) → XDrake → Ace → Robin → Nami → Sanji → Katakuri.
  • Ordre indicatif — nuits suivantes : Aramaki → Kuma → Sengoku → Sabo → Chopper → Moria → Katakuri → Brook (si mort).
☀️ Phase de jour
  • Au lever du jour, tous les joueurs se rassemblent dans la salle du conseil pour l'annonce du MJ : mort(s) de la nuit ou nuit calme.
  • Après l'annonce, les joueurs sont libres de se balader sur la map pour discuter, négocier et former des alliances.
  • En fin de journée, tout le monde retourne dans la salle du conseil pour la phase de débat et de vote.
🗳️ Vote et exécution
  • Dans la salle du conseil, chaque joueur vivant peut accuser et nommer un autre joueur. Le nommé peut se défendre.
  • Un vote à la majorité des joueurs vivants est nécessaire pour aboutir à une exécution. Une seule exécution par jour.
  • Les joueurs morts peuvent toujours parler mais ne votent qu'une seule fois après leur mort.
  • Si aucune majorité n'est atteinte ou si plusieurs joueurs ont le même score, personne n'est exécuté ce jour.
  • Sabo peut ajouter un vote supplémentaire contre une cible via son action de la nuit précédente.
  • Roger peut provoquer une exécution immédiate si un Pirate le nomme — mais uniquement si le nominant est bien Pirate.
  • Si Dragon est exécuté lors du vote, les Pirates perdent instantanément.
  • Après le vote, les joueurs retournent dans leur chambre pour la nuit suivante.
ℹ️ Informations et rôles spéciaux
  • Ivankov peut apparaître comme n'importe quel rôle, y compris Marine, pour les rôles à info.
  • Usopp reçoit toujours de fausses infos correspondant à son rôle de façade Pirate.
  • XDrake possède un rôle de façade Pirate qui change chaque nuit — les autres joueurs peuvent donc avoir des infos contradictoires sur lui.
🎮 Modpack — Obligatoire
  • Le modpack est indispensable pour jouer : sans lui, il est impossible de rejoindre la partie.
  • Il inclut le chat vocal intégré utilisé pendant les phases de nuit et de jour, ainsi que des éléments de décoration pour la map.
  • Suis le tutoriel correspondant à ton launcher ci-dessous.
🔨 Installation — CurseForge
  • Télécharge le fichier ci-dessous.
  • Ouvre CurseForge → clique sur Import.
  • Sélectionne le fichier .zip téléchargé.
  • Lance le modpack depuis ta liste de profils et rejoins la partie.
⬇ Télécharger le fichier CurseForge (.zip)
🚀 Installation — TLauncher
  • Télécharge le fichier ci-dessous.
  • Ouvre TLauncher → clique sur TL Mod → Créer.
  • Donne un nom au choix, sélectionne la version 1.20.1, ne touche pas à la version du chargeur → Créer.
  • Clique sur l'icône 🔧 (tournevis + clé) → Installation manuelle → sélectionne le .zip téléchargé → Installer.
  • Clique sur la flèche à côté du nom du modpack → 🔧 → Ouvrir le dossier du modpack.
  • Ouvre le dossier mods → décompresse le .zip dedans → puis supprime le .zip.
  • Lance le modpack et rejoins la partie.
⬇ Télécharger le fichier TLauncher (.zip)
